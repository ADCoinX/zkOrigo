# zkOrigo - MVP v3

zkOrigo is a **multi-chain stateless compliance validator**.

## Features
- Multi-chain wallet validation
- AI heuristic risk scoring
- zkProof (dummy hash)
- ISO 20022 XML export
- RWA module: asset reserve vs token supply

## Endpoints
- POST /api/validate
- POST /api/rwa_proof
- GET /api/stats
